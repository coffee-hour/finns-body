# Cypher Terminal v11.0.0

High-Fidelity Neural Mesh with Sketchfab Integration.

## Technical Specifications
- **3D Core:** Integrated the "Male Head" Sketchfab model as the central neural mesh.
- **Embedded Optimization:** Utilized Sketchfab embed parameters (`autostart`, `transparent`, `ui_controls=0`) for a native, clean integration.
- **Chromatic Cyber-Yellow Filter:** Applied a sophisticated multi-stage CSS filter (`sepia`, `saturate`, `hue-rotate`, `drop-shadow`) to the iframe to force the 3D model into the Cypher Core neon yellow aesthetic.
- **Interactive UI:** The 3D core reacts to terminal input with real-time brightness and saturation pulses.
- **Voice Protocol:** Professional hacker persona. Zero slang.
